# markdown笔记转为带模板的组会ppt

## 动机
个人习惯用markdown来记录笔记，但是组会汇报又需要固定的模板，一遍遍复制过去太耗时间，于是需要寻找一个高效的md转换为组会ppt的方法

## 步骤

1. 安装Marp for VSCode
2. 看其他教程学习Marp基础用法
3. 在.vscode中注册自定义主题
4. 编写自定义theme的.css文件
5. 享受效率吧！
